# Sistema de Reservación de Hotel

## Descripción

Este proyecto es un sistema básico de gestión de reservas de habitaciones de hotel. Permite realizar reservas, cancelar reservas y mostrar un reporte de las habitaciones disponibles y ocupadas, así como el total a pagar por cada cliente.

## Funcionalidades

1. Reservar una habitación: Permite al cliente reservar una habitación, ingresando su nombre, el número de noches y el precio por noche.


2. Cancelar una reserva: El cliente puede cancelar su reserva liberando la habitación.


3. Ver habitaciones disponibles y ocupadas: Muestra un reporte de las habitaciones disponibles y ocupadas, junto con el total a pagar por cada reserva.

## Instrucciones de uso

1. Reservar una habitación: Selecciona la opción 1. Ingresa el número de la habitación, el nombre del cliente, el número de noches, y el precio por noche.

2. Cancelar una reserva: Selecciona la opción 2. Ingresa el número de la habitación cuya reserva deseas cancelar. El sistema cancelará la reserva y marcará la habitación como disponible.

3. Ver el estado de las habitaciones: Selecciona la opción 3 para ver el reporte de las habitaciones ocupadas y disponibles.

## Estructura del Proyecto

Hotel.java: Archivo principal que contiene la lógica para la gestión de reservas del hotel.
